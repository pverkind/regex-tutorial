2. Other literal characters

As we have seen before, all characters except the 14 metacharacters are literal characters, and they can be used in exactly the same way as the letters in the first exercise.

In the search box below the sample strings, write a pattern that matches the sequence of characters that is common to all three sample strings.

Exercise 2: match the common part of each sample string
Task Text
