Regex tutorial for humanists

5. Word and digit characters (continued)

In some implementations of regular expressions, \d and \w will match not only any digit and word character in Latin script, but in any script represented in Unicode (for example, also in Arabic or Chinese). That's the case in Python and R, for example. In the javascript regular expressions library used for this tutorial (XRegExp), \d matches only the Latin-script (ASCII) digits 0 to 9, but another regular expression, \p{N} (with the "N" of "Numeric") will match any single digit, in any script. Similarly, \p{L} (with the "L" of letter), will match any single letter character, in any script (but not digits, nor underscore!). This also works in Python's regex library.
