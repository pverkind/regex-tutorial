6. Whitespace

Whitespace is a special category of characters, and a number of symbols denote specific types of whitespace in regular expressions:

  • \t : the (horizontal) tab character
  • \n : the newline (aka "linefeed") character
  • \r : the carriage return character (in Windows, new lines are indicated by a combination of carriage return + newline characters: "\r\n"; on Mac and Linux, only the newline character "\n" is used to indicate new lines in a text). This character is a leftover from the typewriter age, in which the "carriage" that held the paper was first returned to its initial position and then the paper was shifted upwards; the result was that the machine could start to write on a new line.
  • \s : any type of horizontal or vertical whitespace (space, tab, newline, carriage return, vertical tab)
  • \S (capital S): any character that is not a whitespace character (that is, the opposite of \s, like \D means the opposite of \d, see lesson 4)
