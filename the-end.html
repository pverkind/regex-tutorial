Regex tutorial for humanists

The end... is only the beginning

This tutorial introduced you to the basic elements of regular expressions. It covers only part of the possibilities of regular expressions.

Useful resources for further learning:

  • Regexone tutorial: very good introduction to basic regular expressions, on which this tutorial is based
  • Regular-expressions.info tutorial: more in-depth tutorial by Jan Goyvaerts, one of the foremost regex experts.
  • Rexegg cheat sheet for all regular expressions patterns, with short explanation and example
  • Virtual keyboard that facilitates writing regular expressions that contain Arabic script.
  • Unicode tables in which you can lookup Unicode characters and copy them easily.
